Cumin-A Scrum Tracker System

Objective: The "Cumin" project is a powerful Scrum-based project management and collaboration tool 
aimed at enhancing productivity for teams by streamlining task tracking, communication, and project 
progress. It offers a centralized platform where project managers and developers can monitor progress, 
assign tasks, and collaborate effectively. The tool is designed to improve workflows and simplify the 
management of multiple projects within an organization.
Key Features -
Task Management:
1. Users can seamlessly add, edit, delete, and track tasks.
2. Mark tasks as "done" or "fixed" with ease to keep workflows smooth.
Discussion Boards:
1. Facilitate project-wide collaboration through integrated discussion boards, promoting better 
communication.
Project Details Management:
1. Manage essential project information such as name, description, deadlines, and types.
2. Track project progress with the ability to mark projects as completed.
Member Management:
1. Add or remove team members.
2. Efficiently assign tasks to specific team members, improving task distribution and accountability.
User Profiles:
1. Create personalized user profiles with bio, skills, and social media links.
2. Profile pictures can be uploaded using Cloudinary, ensuring a streamlined profile management 
experience.

Hardware Requirements:
1. Processor: Minimum Intel i5/AMD Ryzen 5; Recommended Intel i7/AMD Ryzen 7 for better performance.
2. RAM: Minimum 8 GB; Recommended 16 GB or higher for multitasking and efficient development.
3. Storage: Minimum 256 GB SSD; Recommended 512 GB SSD or higher for faster operations.
4. Display: Full HD (1920x1080) or higher for optimal UI/UX development.
5. Graphics Card: Integrated GPU is sufficient, but dedicated GPU (e.g., NVIDIA GTX 1650) is 
recommended for better performance.
Software Requirements:
1. Operating System: Windows 10/11, macOS, or Linux (Ubuntu recommended).
2. Code Editor: Visual Studio Code with extensions like Tailwind CSS IntelliSense and ESLint.
3. Frontend Tools: Tailwind CSS, JavaScript (ES6+), HTML5, CSS3 for UI development.
4. Backend Tools: Node.js and Express.js for backend and API development.
Features
1. Project Overview: Offers a snapshot of the entire project, its progress, and relevant discussions.
2. Task and Sub-task Management: Visual representation of to-do lists, allowing users to track incomplete 
and completed tasks.
3. Bug Tracking: Helps track bugs and their status, enabling seamless updates and resolutions.
4. User Management: Displays a list of project members and their roles.
5. Login & Dashboard: Features secure login with an intuitive dashboard for easy project navigation.
Conclusion:
Cumin is a robust project management solution tailored to the needs of teams aiming to streamline their 
operations, enhance collaboration, and boost productivity. Its key features, such as task tracking, user 
management, and flexible role management, make it an essential tool for businesses to organize and 
execute projects effectively.
